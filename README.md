# Uralog

Uralog is a personal blog about frontend development, programming, and the things I learn along the way.

- Website: https://uralog.me/
- RSS: https://uralog.me/feed.xml
- Atom: https://uralog.me/atom.xml

---

## About this blog

- Notes on frontend development with React, Next.js, and TypeScript
- Mix of technical write-ups and day-to-day learnings
- Aggregates local Markdown posts with Zenn articles

---

## Tech stack

- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **UI**: React, Tailwind CSS
- **Content**: Markdown, Zenn RSS
- **Testing**: Vitest, React Testing Library

---

## Development commands

```bash
# Install dependencies
npm install

# Development server
npm run dev

# Production build & start
npm run build && npm start
```
