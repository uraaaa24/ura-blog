# Hello World!

Welcome to **my blog**! This is a sample post created using MDX. Here, we can combine **Markdown** with **React components** for more flexibility and functionality.

## Features of MDX

MDX allows you to:

- Write rich content with Markdown
- Embed React components directly
- Enhance the interactivity of your site

Here’s an example of a custom React component:

<MyButton label="Click me" />

### Code Example

You can also include code snippets like this:

```js
import React from 'react'

const MyButton = ({ label }) => {
  return <button>{label}</button>
}

export default MyButton
```
